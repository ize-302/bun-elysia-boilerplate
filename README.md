Boilerplate to easily get you started with <a href="https://bun.sh">Bun</a> and <a href="https://elysiajs.com">Elysia framework</a>
</p>

---

#### Usage

1. Clone this reposioty
2. Install dependencies: `bun install`

#### Commands

###### Build project

```bash
bun run build
```

###### Serve build

```bash
bun start
```

###### Run in development mode

```bash
bun start:dev
```
